# Basketball Tracker

## Overview
Basketball Tracker is a full-stack NBA statistics web application designed to track player performance, display advanced metrics, and generate resume-ready stat summaries.

This project is being built with production-quality standards in mind to demonstrate real-world software engineering practices.

## Goals
- Search for NBA players
- Display live game and season statistics
- Store and manage player data
- Generate exportable stat summaries
- Build a clean and responsive user interface
